## 혐오 표현 판별 결과

아래는 입력된 글에 대한 각 카테고리별 판별 결과입니다.

* **(여성/가족:1)**  : "애낳은년들" 이라는 표현은 여성을 비하하고, 출산 경험을 통해 여성의 신체적 변화를 부정적으로 묘사하며,  "소녀스러움"과 대비시켜 여성성에 대한 고정관념을 강화합니다.
* **(남성:0)** : 해당 글에는 남성에 대한 언급이나 혐오 표현이 없습니다. 
* **(성소수자:0)** : 해당 글에는 성소수자에 대한 언급이나 혐오 표현이 없습니다. 
* **(인종/국적:0)** : 해당 글에는 인종이나 국적에 대한 언급이나 혐오 표현이 없습니다. 
* **(연령:1)** :  "어려도"라는 표현과 함께  "애낳은년들" 이라는 표현은 출산을 통해 여성의 나이를 강조하고, 연령에 따른 여성의 가치를 평가절하하는 혐오 표현입니다.
* **(지역:0)** : 해당 글에는 특정 지역에 대한 언급이나 혐오 표현이 없습니다.
* **(종교:0)** : 해당 글에는 종교에 대한 언급이나 혐오 표현이 없습니다.
* **(기타혐오:0)** : 해당 글에는 다른 카테고리에 속하지 않는 혐오 표현은 없습니다.
* **(악플/욕설:1)** :  "애낳은년들", "벗겨보면", "보지건 배때지건", "짼자국" 등의 표현은 욕설 및 여성의 신체를 비하하는 표현으로 악플에 해당합니다. 
* **(Clean:0)** : 해당 글은 혐오 표현과 욕설을 포함하고 있어 Clean 카테고리에 속하지 않습니다.
