* **(여성/가족:0)**  : 여성이나 가족에 대한 직접적인 언급이 없으므로 해당 카테고리에 속하지 않습니다.
* **(남성:0)** : 남성을 특정하여 비하하는 표현이 아니므로 해당 카테고리에 속하지 않습니다.
* **(성소수자:0)** : 성소수자에 대한 언급이 없으므로 해당 카테고리에 속하지 않습니다.
* **(인종/국적:0)** : 특정 인종이나 국적을 지칭하는 표현이 아니므로 해당 카테고리에 속하지 않습니다.
* **(연령:0)** : 특정 연령대를 지칭하는 표현이 아니므로 해당 카테고리에 속하지 않습니다.
* **(지역:0)** : 특정 지역을 지칭하는 표현이 아니므로 해당 카테고리에 속하지 않습니다.
* **(종교:0)** : 종교에 대한 언급이 없으므로 해당 카테고리에 속하지 않습니다.
* **(기타혐오:1)** :  "재기"라는 표현은 극단적인 혐오 표현으로, 특정 집단을 겨냥하지 않더라도 혐오적인 의도를 담고 있으므로 해당 카테고리에 속합니다.
* **(악플/욕설:1)** :  "싹다" 와 "재기"는 욕설에 해당하며, 타인에게 불쾌감을 주는 표현이므로 해당 카테고리에 속합니다.
* **(Clean:0)** : 혐오 표현 및 욕설이 포함되어 있으므로 해당 카테고리에 속하지 않습니다.
