* **(여성/가족:1)**  : "더치페이" 는 한국 사회에서 여성에게 성차별적인 방식으로 사용되는 경우가 많습니다. 여성이 데이트 비용을 남자에게 요구하거나,  남성과 동등하게 비용을 분담하려는 행위를 비난하는 데 사용되는 경우가 많기 때문입니다. 이 문장은  "더치페이"를 부정적으로 묘사하며, 마치 여성이 더치페이를 요구하는 행위를 비난하는 듯한 어투를 사용하고 있습니다. 
* **(남성:0)** : 남성을 직접적으로 비하하거나 조롱하는 표현은 없습니다.
* **(성소수자:0)** : 성소수자를 언급하거나 혐오하는 표현은 없습니다.
* **(인종/국적:0)** : 특정 인종이나 국적에 대한 언급이 없습니다.
* **(연령:0)** : 특정 세대나 연령을 비하하는 표현은 없습니다.
* **(지역:0)** : 특정 지역에 대한 언급은 없습니다.
* **(종교:0)** : 종교에 대한 언급은 없습니다.
* **(기타혐오:0)** :  장애인, 정부, 기자, 경찰 등에 대한 혐오 표현은 없습니다.
* **(악플/욕설:0)** : 비하나 욕설은 포함되어 있지 않습니다.
* **(Clean:0)** :  혐오 표현이 포함되어 있으므로 Clean 카테고리에 속하지 않습니다.
