## 판별 결과

해당 글은 다음과 같이 분류됩니다.

* **(여성/가족:1)**  : "제 아내가, 딸이 저런 일을 당한다면" 이라는 표현은 여성을 대상으로 한 혐오 표현이라고 판단할 수 있습니다.  
* **(남성:0)** : 남성에 대한 혐오 표현은 없습니다.
* **(성소수자:0)** : 성소수자에 대한 혐오 표현은 없습니다.
* **(인종/국적:0)** : 인종이나 국적에 대한 혐오 표현은 없습니다.
* **(연령:0)** : 연령에 대한 혐오 표현은 없습니다.
* **(지역:0)** : 지역에 대한 혐오 표현은 없습니다.
* **(종교:0)** : 종교에 대한 혐오 표현은 없습니다.
* **(기타혐오:0)** :  기타 혐오 표현은 없습니다.
* **(악플/욕설:0)** : 악플이나 욕설은 없습니다.
* **(Clean:0)** :  Clean 카테고리에 해당하지 않습니다.
