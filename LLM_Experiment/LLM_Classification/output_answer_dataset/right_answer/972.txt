* **(여성/가족:0)**  : 여성이나 가족에 대한 혐오 표현은 없습니다.
* **(남성:0)** : 남성에 대한 혐오 표현은 없습니다.
* **(성소수자:0)** : 성소수자에 대한 혐오 표현은 없습니다.
* **(인종/국적:0)** : 인종이나 국적에 대한 혐오 표현은 없습니다.
* **(연령:0)** : 연령에 대한 혐오 표현은 없습니다.
* **(지역:0)** : 특정 지역에 대한 혐오 표현은 없습니다. "반지하"는 주거 형태를 나타내는 표현이며, 특정 지역을 지칭하는 표현이 아닙니다.
* **(종교:0)** : 종교에 대한 혐오 표현은 없습니다.
* **(기타혐오:0)** :  기타 혐오 표현은 없습니다.
* **(악플/욕설:0)** : 욕설이나 비하 표현은 없습니다.
* **(Clean:1)** : 혐오 표현이나 욕설이 없는 일반적인 문장입니다.
